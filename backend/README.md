# Backend API

## Scripts

### Development
- `npm run dev`: Start development server with nodemon
- `npm start`: Start production server

### Code Quality
- `npm run lint`: Run ESLint check
- `npm run format`: Format code with Prettier
- `npm test`: Run tests with Jest

## Technology Stack
- Node.js & Express
- MongoDB (Mongoose)
- Socket.io
- Razorpay
