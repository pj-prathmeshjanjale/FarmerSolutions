#  Agriculture Platform Frontend

A modern, responsive React-based dashboard for farmers and agriculture sellers.

##  Features

- **Personalized Dashboards**: Tailored views for Farmers and Sellers.
- **Equipment Marketplace**: Browse, list, and rent farming machinery.
- **AI Farming Assistant**: Interactive chatbot for agriculture-related queries.
- **Weather Widget**: Localized weather forecasts for better planning.
- **Real-time Chat**: Connect with buyers and sellers instantly.
- **Secure Auth**: Role-based access control with JWT.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router 7
- **Communication**: Socket.io Client
- **API Client**: Axios
